Amazon Interview | Set 30

I have attended the interview for Software Development Engineer position and got the offer from Amazon.

I spent a lot of time in GeeksforGeeks going through the precise and simple explanations of complex problems, which helped me to sharpen my mind. Really, its a great work by the geeks and I am happy that I am a part of it.

The following were the questions.

Round 1 : Write a program to solve the below problems. (Time 1.30 hrs)(Written Test)
1. Given a string in the form of a Linked List, check whether the string is palindrome or not. Don’t use extra memory. Give the time complexity. The node structure is 

Class Node {
          Char data;
          Node next;
}

2. Given a Binary Search tree along with the parent pointer, find the next largest node for the given node. Give the time and space complexity. The node Structure is

                class Node {
                     Int data;
                     Node left;
                     Node right;
                     Node parent;
                }

3. Given a sorted array which is rotated n number of times. Find out how many times the array is rotated. Time complexity should be less than O(n).

Round 2 : With Team Member
1. Tell me about yourself.

2. Explain your project.

3. Given a Binary tree, find the vertical sum.
….a. I gave a solution using hashmap. There were discussion about the problems (time and space complexity) in using hash map. Then due to its cons, he told me to use some other DS to solve the problem.
….b. Then I gave a solution using Array. There were discussion about how it can be used, time and space complexity and its pros and cons.
….c. Code using Array.

4. Given a matrix mxn, where all the rows were sorted, print the elements in the matrix in a sorted order.
….a. I gave a solution with O(mxmxn) time complexity.
….b. He wanted a solution in O(mnlog(m)) time complexity and gave a hint to use heap.
….c. Code for the same.

Round 3: With 3rd Level Manager (culture Fit)
1. Tell me about yourself.

2. Explain Your accomplishments.

3. What you are proud of yourself?

4. How you will handle the conflict with the team member?

5. Lot of behavior oriented questions.

6. Given a String, remove the duplicates in the string.
….a. Lot of variations from the same problem.
….b. Asked for a solution in different time and space complexities and the complications involved.
….c. I guess the communication skill might have been tested here.

7. Given a floating point number, write a program to convert it into a string. The number of digits after decimal point can be more than 1000.

Round 4 : With Manager
1. Can you tell me about yourself?

2. Explain the projects you worked on?

3. Given a Binary tree, connect all the leaf nodes in the form of a doubly linked list. Don’t use extra space.

4. A scenario was given about two robots and its functionality. Write a program which will be running in both the robots which will perform the specified functionality.

5. Given an integer, find the next largest integer using the same digits as in the given integer. For example, if 12345 was given, the program should return 12354.

Round 5 :
1. Tell me about yourself.

2. Explain what you have done in your previous company.

3. As I have worked on a product and they told to explain the product

4. What are the developments you have done and what impact it will be having?

5. What will happen to your development, if the product Is migrated?

6. Questions on threading.

7. What is a thread safe code? Explain.

8. What is a process and thread? Differences?

9. Given a binary tree print the elements in a zig zag order.

Thanks a lot for Geeks team.

void checkPalindromeProperty(Node* first) {
	int size=getLinkedListSize(first);
	if(size==0) {
		printf("\nERROR: Input string is empty!\n\n");
	} else if(size==1) {
		printf("\n\nIt's a Palindrome String.\n\n");
	} else {
		int i;
		int palindrome=1;	// Flag variable
		Node* mid=NULL;
		Node* last=NULL;
		Node* p=NULL;
		Node* q=NULL;
		mid=first;
		// Set mid pointer
		for(i=0;i<(size+1)/2;i++) {
			mid=mid->next;
		}
		// Reverse the links of nodes from middle to last
		p=mid->next;
		mid->next=NULL;
		last=mid;
		for(i=0;i<size/2-1;i++) {
			q=p->next;
			p->next=last;
			last=p;
			p=q;
		}
		// Save the last pointer for reuse
		q=last;
		// Compare characters from both the ends
		for(i=0;i<size/2;i++) {
			if(first->data != last->data) {
				palindrome=0;
				break;
			}
			first=first->next;
			last=last->next;
		}
		if(palindrome) {
			printf("\n\nIt's a Palindrome String.\n\n");
		} else {
			printf("\n\nNot a Palindrome String!\n\n");
		}
		// Set the links back to its original state
		last=q; // Restore the last pointer
		p=last->next;
		last->next=NULL;
		for(i=0;i<size/2-1;i++) {
			q=p->next;
			p->next=last;
			last=p;
			p=q;
		}
	}
}

      #define M 5
      #define N 4

      void swap(int heap[][N],int index1,int index2,int col) {
      int temp=heap[index1][col];
      heap[index1][col]=heap[index2][col];
      heap[index2][col]=temp;
      return;
      }

      void heapify(int heap[][N],int index,int size,int col) {
      int max=-1;
      if( 2*index+1 < size && heap[index][col] < heap[2*index+1][col]) {
      max=2*index+1;
      }

      if(2*index+2 < size && heap[index][col] < heap[2*index+2][col]) {
      if(heap[max][col] < heap[2*index+2][col])
      max=2*index+2;
      }

      if(max!=-1) {
      swap(heap,index,max,col);
      heapify(heap,max,size,col);
      }

      }

      void buildHeap(int arr[][N],int size,int col) {

      for(int i=(size-1)/2;i>=0;i–) {
      heapify(arr,i,size,col);
      }
      }

      void heapsort(int heap[][N],int size,int col) {

      if(size==1)
      return;
      swap(heap,0,size-1,col);
      heapify(heap,0,size-1,col);

      heapsort(heap,size-1,col);
      }

      void sortMatrix(int arr[][N]) {
      for(int i=0;i<N;i++) {
      buildHeap(arr,M,i);
      }

      for(int i=0;i<N;i++) {
      heapsort(arr,M,i);
      }

      }

      void printSortedMatrix(int arr[][N]) {
      for(int i=0;i<M;i++) {
      for(int j=0;j<N;j++) {
      std::cout<<arr[i][j]<<" ";
      }
      }
      }

/*
	Purpose: Find out the next node larger than the given one in a binary search tree
	Time : O(n)
	Space: O(1)
*/

#include<stdio.h>
#include<stdlib.h>

typedef struct BST {
	int data;
	struct BST* parent;
	struct BST* left;
	struct BST* right;
}Node;

// Return the node pointer of the given number in this BST

Node* getNode(Node* root, int n) {
	if(root==NULL) {
		printf("\n\nERROR: Required node doesn't exist into the given BST.");
		return NULL;
	}
	if(root->data == n) {
		return root;
	} else if(root->data < n) {
		getNode(root->right, n);
	} else {
		getNode(root->left, n);
	}	
}


// Find out next larger node than the given node

Node* getNextLargerNode(Node* root, int n) {
	Node* node=getNode(root,n);
	if(node==NULL) {
		return NULL;
	}
	// If this node is leaf node, then try to traverse the parent branch
	// Otherwise, try to traverse the leftmost branch

	if(node->right==NULL) {
		node=node->parent;
		while(node) {
			if(node->data > n) {
				return node;
			}
			node=node->parent;
		}
	} else {
		Node* lastVisited=node;
		node=node->right;
		while(node) {
			lastVisited=node;
			node=node->left;
		}
		return lastVisited;
	}
}

// Insert this newly created node into BST specified by root node

void insertNode(Node* root, Node* node) {
	if(root==NULL || node==NULL) {
		printf("\n\nERROR: Either root or node is NULL!");
		return;
	}
	if(root->data == node->data) {
		// Since argument node already exists in the given BST, no need to insert it again
		printf("\n\nINFO: Node to be inserted already exists in the given BST.");
		return;
	} else if(root->data < node->data) {
		if(root->right == NULL) {
			root->right=node;
			node->parent=root;
			return;
		} else {
			insertNode(root->right, node);
		}
	} else {
		if(root->left == NULL) {
			root->left=node;
			node->parent=root;
			return;
		} else {
			insertNode(root->left, node);
		}
	}
}

// Create new BST node which needs to be inserted into the existing BST

Node* createNode(int n) {
	Node* node=(Node *)malloc(sizeof(Node));
	node->data=n;
	node->parent=NULL;
	node->left=NULL;
	node->right=NULL;
	return node;
}

// Create new binary search tree from set of input numbers

Node* createBST(void) {
	int i=1;
	int n;
	Node* root=NULL;
	while(1) {
		printf("\n(Enter -1 to finish) Enter number: ");
		scanf("%d",&n);
		if(n==-1) {
			break;
		}
		if(i==1) {
			root=createNode(n);
		} else {
			insertNode(root,createNode(n));
		}
		i++;
	}
	return root;
}

// Print this BST in inorder fashion

void print(Node* root) {
	if(root) {
		print(root->left);
		printf("%d ",root->data);
		print(root->right);
	}
}

// Main function

int main(){
	Node* root=createBST();
	Node* nextNode=getNextLargerNode(root,70);;

	printf("\n\n");
	print(root);
	printf("\n\n");

	// TEST: Find out the next node larger than 70 from the previously created BST

	printf("\n\nNext node larger than 70 = %d\n\n",nextNode?(nextNode->data):-1);
	return 0;
}

/*
	Purpose: Check palindrome property of the string defined as a linked list
	Time : O(n^2)
	Space: O(1)
*/

#include<stdio.h>
#include<stdlib.h>

typedef struct LinkedList {
	char data;
	struct LinkedList* next;
}Node;

// Insert new node and return its address
Node* insert(char data) {
	Node* last=(Node *)malloc(sizeof(Node));
	last->data=data;
	last->next=NULL;
	return last;
}

Node* create(void) {
	char ch;
	int i=1;
	Node* first=NULL;
	Node* last=NULL;
	while(1) {
		printf("\n(Enter '1' to finish) Enter char: ");
		scanf("%s",&ch);
		if(ch=='1') {
			break;
		}
		if(i==1) {
			last=first=insert(ch);
		} else {
			last=last->next=insert(ch);
		}
		i++;
	}
	return first;
}

void print(Node* node) {
	printf("\n");
	while(node) {
		printf("-> %c ",node->data);
		node=node->next;
	}
}

int getLinkedListSize(Node* first) {
	int size=0;
	while(first) {
		first=first->next;
		size++;
	}
	return size;
}

void checkPalindromeProperty(Node* first) {
	int size=getLinkedListSize(first);
	if(size==0) {
		printf("\nERROR: Input string is empty!\n\n");
	} else {
		int i,j;
		Node* last=NULL;
		for(i=size;i>0;i-=2) {
			last=first;
			for(j=0;j<i-1;j++) {
				last=last->next;
			}
			if(first->data != last->data) {
				printf("\n\nNot a Palindrome String!\n\n");
				return;
			}
			first=first->next;
		}
		printf("\n\nIt's a Palindrome String.\n\n");
	}
}

int main(){
	Node* first=create();
	print(first);
	checkPalindromeProperty(first);
	return 0;
}

      @Hitesh,

      You can solve this in O(n) time if linked list is not read-only.

      Hint:
      1. Find the middle of the list
      2. Reverse the linked list from middle to end
      3. Have 2 pointers one starts at head and other starts at mid
      4. start comparing both the pointers by traversing the list
      5. Reverse the list again from the middle to end to retain the original list
      Note: Take care of odd/even size of the list.
