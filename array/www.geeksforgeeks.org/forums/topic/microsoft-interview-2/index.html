    Hello All,

    I would like to share my experience with Microsoft IDC, Hyderabad. This was for Bing team. I hope this would be of some help to those who are preparing for Microsoft.

    Note: All the rounds start with a discussion about the candidate and his past work. They also ask if the candidate has any preferences in the roles like development/testing etc. At the end of each round, candidate is asked if there are any questions for the interviewer.

    A. Screening over Skype (as I was in Bangalore)
    1. Given a Linkedlist where each node has a next and random pointer, create a clone of this list
    2. What data structure is used to represent a file system in UNIX
    3. write a program to check if a given tree has any cycle

    After this, I was made to fly to Hyderabad.

    B. Face to Face at Hyderabad
    1. Given a tree, where each node may have more than two children, write a program to print the zig-zag level traversal of the tree
    2. Given a large 2-D matrix containing integers, find the mean of any given submatrix in a constant time (no constraints on space and preprocessing of this matrix)
    3. How to identify malicious results when you are using google search or bing search

    c. Face to Face at Hyderabad
    1. Diameter of a binary tree

    Unfortunately, I didn’t make it through the second round at Hyd. But it was a good experience interviewing with them.

    All the best for those who are going to interview!!

    B-2 is a simple inclusion-exclusion principle problem.
    Precalculate sums of 2D-prefixes – all submatrices with left-upper corner (0,0). Then, on each query, calculate sum of a given submatrix (x1,y1),(x2,y2) in O(1):

    sum = sums[x2, y2] – sums[x2, y1] – sums[x1, y2] + sums[x1, y1]

    And, finally find the mean dividing sum by the number of elements in the submatrix, which is simple:

    mean = sum / ((x2 – x1) * (y2 – y1))
