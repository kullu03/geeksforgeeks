    Do a BFS of the tree till you find any one of the nodes.

    Then keepthing this node as root, perform DFS for the other node.

    - If the two nodes are on same level, you will find one node before the other and DFS will fail while searching for other.

    – If nodes on different levels, you will find one node before other

    Subcase a) If there exists a path between two, you will find second node while doing DFS

    Subcase b) If there doesnt exist a path between two, DFS will terminate without finding second node.
