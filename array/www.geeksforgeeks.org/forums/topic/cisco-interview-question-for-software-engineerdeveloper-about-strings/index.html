    char ch[]=”hello.”:–is a constant pointer to a non-constant string..we can not point ch to another address..or we can not try to change the address of ch …whereas we can change the contents of the array..

    char * ch=”hello”;—is a non-const pointer to a non-const data…we can make ch to point to some other memory location…but when we will try to change the content through ch ,then behave is UNDEFINED…and on some compiler runtime error will be there…

    c1++;//not allowed
    while c1[2]=’g’//allowed
