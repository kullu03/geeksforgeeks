    along with the circular queue, take an extra variable ‘sum’ which maintains the sum of hits.

    After every second update the queue and sum.

    (1) update sum, remove deque element and add ith hits.

    sum = sum – dequeue() + num(to be enqued at ith sec.)

    (2) enque(num). //which overwrites the (i-60)th value.

    this ensures that all the operations can be done in O(1) time.
