    Consider the Fibonacci Seies 1, 2, 3, 5, 8, 13…..

    Now consider a Fibonacci binary number system of numbers where numbers are considered as sum of Fibonacci numbers

    Ex. 10 can be written as 8+2=>10010 and 17 can be written as 13+3+1 => 100101. This representation is similar to binary representation. Instead of powers of 2, Fibonacci numbers are used.

    The question was, given n find all possible representations of n in Fibonacci binary number system.

    Here is one approach.

    1. Generate fibonacci series till number is just greater than it.

    2. Subtract the number from just smaller fibonacci number.

    3. Binary search rest of the subsets in already calculated fibonacci series & update solution vector simultaneously.

    4. continue till number is not zero.

    public static ArrayList> findAll (int target) {
    ArrayList    > results = new ArrayList>();
    // you should generate a fibo array according to your need
    int[] fibo = {1, 2, 3, 5, 8, 13};
    doFindAll(fibo, target, 0, new ArrayList(), results, 0);
    return results;
    }

    public static void doFindAll (int[] fibo, int target, int position, ArrayList result, ArrayList> results, int sum) {
    if (position == fibo.length) {
    if (sum == target) {
    results.add((ArrayList)result.clone());
    }
    } else {
    // case1: do not choose current num
    doFindAll(fibo, target, position+1, result, results, sum);

    // case2: choose current num if we can
    if (sum + fibo[position] == target) {
    result.add(fibo[position]);
    results.add((ArrayList)result.clone());
    result.remove(result.size()-1);
    } else if (sum + fibo[position] < target) {
    result.add(fibo[position]);
    doFindAll(fibo, target, position+1, result, results, sum+fibo[position]);
    result.remove(result.size()-1);
    }
    }
    }
