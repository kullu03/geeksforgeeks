    You are given weights of order 1kg 3kg 9kg 27kg 81kg and so on…(powers of 3)

    A weight of N kg is placed on any one of the empty pans of the balance.

    You are supposed to add weights to both pans of the balance and make them equal.

    You can use one weight only once..i.e., you can’t use 2 number of 3kg weights.

    ps: Here, N belongs to a set of postive integers. and of course N is not a power of 3

    Question is not clear.

    We can also make such counting using powers of 2. Every integer can be expressed in powers of 2.

    If we have weights 1, 2, 4, …. 2^r, we can measure any weight upto S(n) = 2^(r+1) – 1. You can check this property of binary system.

    Also powers of 3 satisfy this property.
