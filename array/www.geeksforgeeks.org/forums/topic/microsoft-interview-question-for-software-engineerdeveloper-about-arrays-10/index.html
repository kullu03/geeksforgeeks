      Simply keep another variable (here “pos”) to keep track of index of nonzero elements in the array.
      1.each time 0 occurs, update the zero count.(count++)
      2.each time 0 doesnt occur, do a[pos++]=a[i]
      3.finally starting from back of the array till count times, store zero repeatedly.
      Requires one traversal and one more to store zeros at the back of the array.

      Code:

      void shiftZeros(int* a, int size)
      {
      int count=0,i,j,pos=0;
      for(i=0;i {
      if(a[i]==0)
      {
      count++;
      }
      else a[pos++]=a[i];
      }

      for(i=0;i {
      a[size-1-i]=0;
      }
      }
