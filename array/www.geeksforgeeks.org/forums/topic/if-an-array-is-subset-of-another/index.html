    Algo to find Array2 is subset of Array1 or not

    Let the lengths of Array1 and Array2 be m and n respectively.

    1) Sort both Array1 and Array2. O(mLogm + nLogn)

    2) Now run KMP algorithm to see if Sorted Array2 is a subarry of Sorted Array1. O(m+n)

    Time Complexity: O(mLogm + nLogn)

    @guest01 :

    Agreed, my above approach doesn’t work for cases like this. Following is the modified one. Although the step 2 is costly now, time complexity is still O(mLogm + nLogn) .

    Algo to find Array2 is subset of Array1 or not

    Let the lengths of Array1 and Array2 be m and n respectively.
    
1) Sort both Array1 and Array2. O(mLogm + nLogn)
    
2) For each element of Array2, do binary search for it in Array1.
    
a) If the element is not found then return 0.
    
3) If all elements are present then return 1.

    Time Complexity: O(mLogm + nLogn)
