    A frog has to cross a river. There are n rocks in the river, using which the frog can leap across the river. On its way across the river the frog can chose to skip a rock, but it cannot skip two consecutive rocks because that would be two far a distance for the frog to hop, also the from would not skip the first rock and the last rock. E.g. if there are 3 rocks, 1,2,3 and 4, there could be three following routes it could take:

    1,2,3,4

    1,2,3,4

    1,3,4

    1,2,4

    Write a recursive algorithm, that takes a number of rocks’ and prints all the feasible paths. Ofcourse there can be other arguments too.

    Simple solution:
    If frog is currently at rock i, then there are 2 possibilities, either he go to (i+1)th rock or (i+2)th rock.We can transform this into binary tree as follows. Start from rock 1 as root of tree. Every node can have at most two children.
    leftChild(root)=root+1
    rightChild(root)=root+2
    Terminate when u reach leaf node.
    Leaf node will be n.

    Now this is equivalent to print all root to leaf paths in binary tree.
    Code :
    `void func(int arr[],int root,int ind,int n){
    if(root>n) return;
    arr[ind]=root;
    if(root==n){
    for(int i=0;i<=ind;i++)
    printf(“%d “,arr[i]);
    printf(“\n”);
    return;
    }
    func(arr,root+1,ind+1,n); // left child
    func(arr,root+2,ind+1,n); // right child
    }
    void printPath(int n){
    int *arr=new int[n+1];
    func(arr,1,0,n);
    delete arr;
    }

    I can’t believe people are so eager to write code without giving out the correct reasoning/explanation. Here is the correct solution involving dynamic programming (duh, the question does give a hint by saying you can add additional args in the recursive function):

    Let F(i,j) := Number of path through which frog can reach ith rock and also touches jth rock midway, where j

    With the above definition,

    F(i,j) = F(i-1, j) + F(i-2, j) – F(i, i-1)

    The reason for above equation is that to reach ith rock through jth rock, there are paths from i-1th rock and i-2th rock. Note that however, path to i-1th rock can be achieved from i-2th rock also. Hence the final deduction term F(i, i-1)
