    Indexed access of arrays are used when we are sure the dimensions of array at compile time. Many places we use runtime calculation of element address, based on row-major storage of array elements in C/C++.

    For example turning an image by 90 deg (http://geeksforgeeks.org/?p=10402). An image dimensions can’t be confirmed at compile time. We need to access the image as 2D array at runtime.

    Another example code here (visualizing the dynamic programming algorithm)

    http://geeksforgeeks.org/?p=12998&cpage=1#comment-4321 (see the display function)

    In C/C++ arrays are accessed in row-major order. Accordingly we need to calculate the offset an element from it’s base. Above two links provide sample examples.
