    A matrix filled with 0’s and 1’s. if u visit a position (i,j) east,west.North,south and that position will be fliped. 0 to 1 and vice versa. Find a way so that they can be filled with all 1’s.

    this might not be the best algorithm but solves the problem. O(n^4);

     

    1. Start from first position. Cursor at starting of the array (0,0)

    2. find the first 0. first 0 is such that it is first if we traverse the array row wise that is (0,0) to (0,n) then (1,0) to (1,n).. and so on. let say it is (i,j)

    3. Move the cursor to (i,j) now path from (0,0) to (i,j)(excluding (i,j))  all are 0.

    4. now move back to (0,0) now path from (0,0) to (i,j) incuding (i,j) all are 1.

    5. till now we have successfully changed one 0 to 1.

    6. Repeat the process till the end of the metrics.
