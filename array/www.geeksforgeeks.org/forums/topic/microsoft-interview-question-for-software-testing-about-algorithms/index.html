    The sorting using that function is fairly trivial. Something you should have cracked. But hey! if you got placed, then who cares?

    The idea is to swap zeorth element and the last element in the positioned at “sizeOfArray” Now just like in selection sort, you have to decide whether you want to “bubble” up the min element or the max element to the end.

    Lets look at an example of descending sort via selection

    size = a.size()
    
max = a[0]
    
max_position = 0
    

    
//iterate through n elements
    
for i = 1 to size {
    
//iterate through n-i elements and find the max
    
for j = i to size {
    
if ( max < a[j] ) {
    
//note max ele and its position down
    
max = a[j];
    
max_position = j;
    
}
    
}
    
//once the max is fixed, bubble it up
    
//to the top (ith position) of the array and forget about it
    
swap(a, size, j)
    

    
//loop for i+1...n
    
}
