    @akshayjohri
    You don’t just need a random number but the one that’s generate random numbers from 1-100(in this case) equally probably. Putting it simply, say there’s a random number generator whose range is say 0-150 then you’re code would indeed give more numbers from 0-50 than 51-99.

    We need a random number generator which would generate numbers (generally has a scaled range) of equal proportions, i.e., would generate numbers equally probably suiting our case.(In this case it is 100, somewhere else it might be some other number say 178, so we should also be careful while choosing the random number generator)
