    Given X’s present age = k yrs
    Let Y’s present age = y yrs
    Let the time period used to compare ages = c yrs

    Now, as per the statement,
    Presently, X is twice as old as Y was c yrs ago, so
    k = 2(y-c) …. (1)
    c yrs ago, X was as old as Y is now, so
    k-c = y
    c = k-y …. (2)
    substituting (2) in (1)
    k = 2(y-k+y)
    k = 2(2y-k)
    k = 4y – 2k
    3k = 4y

    Hence the ratio is 4:3
