    Online Test
    1. Given a binary tree, find if there exists a root to leaf path of distance k.
    2. Given two numbers, where each digit is represented as a node of a linked list, add those numbers

    Written Test
    1. Given an array of digits ranging [0-9]. Form a maximum number which is divisible by 2,3, and 5
    2. Given binary tree, print the nodes which are at k distance from root
    3. Given a linked list, reverse every three nodes of the linked list
