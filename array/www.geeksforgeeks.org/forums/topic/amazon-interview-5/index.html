    O(n2) approach!

    Generate all possible string and build a trie with these string.

    It is easy to track in a trie when a string comes once or more than once.

    A sub string can start and end at any location in the array.

    So basically a sub string can be represented by S(i,j) where 0 <= i <j < n.

    Total pairs of (i,j) is nC2

    Time complexity O(n2)

    Space complexity O(n2)
