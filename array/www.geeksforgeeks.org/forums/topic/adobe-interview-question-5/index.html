    Given N points(in 2D) with x and y coordinates… Find a point P (in N given points) such that the sum of distances from other(N-1) points to P is minimum.

  • April 25, 2012 at 10:51 AM #112004

    Following is a simple O(n^2) solution.


    
1) min_distance_sum = INT_MAX
    
2) Iterate through all of the points and do following for every point p.
    
a) Calculate sum of distances of all other n-1 nodes from p.
    
b) If this distance is smaller than min_distance_sum, then update min_distance_sum.
    
3) Return min_distance_sum
