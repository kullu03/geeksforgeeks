    k distinct integers [0, N)

    Select a random no [0,N) which is not in this k distinct list.

    Example:

    [4, 6, 9]

    Choose a random no between 0 – 9 which is not 4, or 6, or 9.

    Valid output: 2

    Invalid output: 6

    This will work –>

    Suppose list of k integers is [4,6,9]

    We know we can randomly choose only from remaining N-k integers.

    So, map [0,N-k) to these remaining N-k numbers.

    In other words imagine a (N-k) sided die painted with only these valid numbers.

    i.e.

    for above example

    Id -> no.

    0 -> 0

    1 -> 1

    2 -> 2

    3 -> 3

    4 -> 5

    5 -> 7

    6 -> 8

    Now choose randomly from id and return corresponding number.

using namespace std;

int random(int a, int b) {
  static int first_run=1;
  if(a==0 && b==1) {
    if(first_run==1) {
      srand( (unsigned)time( NULL ) );
      first_run=0;
    }
    return rand()%2;
  } else if(a==b){
    return a;
  } else if(b==a+1) {
    if(random(0, 1)) {
      return b;
    } else {
      return a;
    }
  } else {
    if(random(0, 1)) {
      return random((a+b)/2+1, b);
    } else {
      return random(a, (a+b)/2);
    }
  }
}
