    Constraints: Values inside the nodes should not be changed. Only pointer manipulation is allowed.

    Take 2 pointers – First(for 0) and second(for 2).

    First is pointing to starting node, Second is pointing to ending node. While traversing the head pointer, three cases are possible :

    CASE 0: if we found a zero, then move that node to the starting of the link list, and the “first” pointer is now pointing to this new zeroth node.

    CASE 1: Do nothing. Leave it as it is.

    CASE 2:Move that node to the end of link list and the “second” pointer is now pointing to this new second node..

    So, we are moving the node containing 0 to starting of the link list, node containing 2 to the end of the list. The node containing 1 is not needed to be moved anywhere, because in the end, we are left with all the 0 in the beginning, 1 in the middle and 2 in the last. So the list is sorted in O(n) time.

    maintain six pointers head1 , tail1 , head2 , tail2 , head3 , tail3
    Suppose *temp is the iterator that is traversing the list.

    1 when temp encounters 0( for the first time) set head1 and tail1 to the same. When another 0 is encountered set tail1->right= temp and tail1=temp;

    Same for the other two. When the traversal will be over , we will have three heads and tails. And then point tail1 to head2 and tail2 to head3. This will be it.

    Complexity O(n).

    void sort0_to_n(node **header)//In this case the numbers are 0,1 and 2(n=2).But this function can work for all ‘n’.
    {
    int i,n=2,k,flag,start;
    node *last=*header,*heads[n],*tails[n];

    for(i=0;i {
    heads[i]=NULL;
    tails[i]=NULL;
    }

    while(last)
    {
    if(heads[last->val]==NULL)
    {
    heads[last->val]=tails[last->val]=last;
    }
    else
    {
    tails[last->val]->next=last;
    tails[last->val]=last;
    }

    last=last->next;
    }

    k=0,flag=0,start=0;
    for(i=0;i {
    if(heads[i]==NULL)
    continue;

    if(flag==0)
    {
    k=i;
    start=i;
    flag=1;
    continue;
    }

    tails[k]->next=heads[i];
    k=i;
    }
    tails[k]->next=NULL;

    *header=heads[start];

    }
    main()
    {
    node *header=read();
    printf(“Original\n”);
    display_list(header);
    sort0_to_n(&header);
    printf(“\nAfter sorting\n”);

    void create(node** headref,node** tailref,node** temp )
    {
    if(!*headref)
    {
    *headref=*temp;
    *tailref=*headref;
    }
    else
    {
    (*tailref)->next=*temp;
    *tailref=*temp;
    }
    }
    void move(node* head)
    {
    node* temp=head;
    while(temp)
    {
    switch(temp->data)
    {
    case 0:
    create(&head0,&tail0,&temp);
    break;

    create(&head1,&tail1,&temp);
