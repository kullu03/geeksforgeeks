    Given a Binary Tree having nodes numbered from 1 to n in level order fashion.

    e.g.

    1

    2 3

    …and so on.

    A 2-D Matrix (nxn) is also given(initially set to 0).

    Now, We have to fill the matrix such that Matrix(i,j)=1 , if ‘i’ is an ancestor of ‘j’.

    Note : The root will have all entries as 1(except Matrix(root,root) : root has no ancestor).

    (Coz, root is the ancestor of all nodes)

    Better approach than O(n2)

    No need to store elements in tree, as it is complete binary tree & elements are sequential. Ancestors can be easily calculated.

    Time complexity: O(n logn)

    void fillMatrix(int n)
    
{
    
int M[n+1][n+1],i,j;
    

    
for(i=0;i<=n;i++)
    
for(j=0;j<=n;j++)
    
M[j]=0;
    

    
for(j=1;j<=n;j++)
    
{
    
i=j>>1;
    
while(i)
    
{
    
M    [j]=1;
    
i>>=1;
    
}
    
}
    

    
for(j=1;j<=n;j++)
    
{
    
printf("Ancestors of %d: ",j);
    
for(i=1;i<=n;i++)
    
if(M    [j])
    
printf("%d ",i);
    
printf("n");
    
}
    
}
    

    
int main()
    
{
    
fillMatrix(7);// 7 number of nodes from 1 to 7
    
return 0;
    
}
