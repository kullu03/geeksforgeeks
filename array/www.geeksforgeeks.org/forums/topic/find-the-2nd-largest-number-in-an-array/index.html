    It can be done using Heap data structure, if the size of array is large. Arrange the array into Max heap.Then Delete the max element which is the root node and then Heapify that gives the second max element as the root node. This heapify takes O(nlogn).

    A quick and simple solution, would be starting with the first element of the array as the max value, and the minimum integer value allow as the second largest – one pass is performed, so complexity is O(n). C# implementation follows:

            static int findsecondMax(int[] m){
    
//Check for empty array
    
if (m.Length == 0) {
    
return int.MinValue;
    
}
    
int max = m[0];
    
int sMax = int.MinValue;
    

    
for (int i = 0; i < m.Length; i++) {
    
if (m > max) {
    
max = m    ;
    
}
    
else if (m     > sMax) {
    
sMax = m    ;
    
}
    
}
    
return sMax;
    
}
