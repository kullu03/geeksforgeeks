    Given N numbers , [N< =10^5] we need to count the total pairs of numbers that have a difference of K. [K>0 and K<1e9]

    Input Format:
    1st line contains N & K (integers).
    2nd line contains N numbers of the set. All the N numbers are assured to be distinct.
    Output Format:
    One integer saying the no of pairs of numbers that have a diff K.

    Sample Input #00:
    5 2
    1 5 3 4 2

    Sample Output #00:
    3
     
    Sample Input #01:
    10 1
    363374326 364147530 61825163 1073065718 1281246024 1399469912 428047635 491595254 879792181 1069262793 

    Can someone give a logic that works in O(N) ?

    Hi! Let’s store a boolean array B of length 1e9 such that B[i] is true if the number i exists in the given array.

    Thus, we can determine in O(1) whether some number exists in the given array or not.

    Then, the problem can be solved in O(N): 

    
#include <iostream>
#include <vector>
using namespace std;

const int MAXN = 1e9;

int main() {
  int n, k, a, ans = 0;
  vector<bool> b(MAXN, false);
  
  cin >> n >> k;
  
  while (n--) {
    cin >> a;

    b[a] = true;

    if (a - k >= 0 && b[a - k])    // if (a-k) exists
      ans++;

    if (a + k < MAXN && b[a + k])    // if (a+k) exists
      ans++;
  }
  
  cout << ans << endl;
}
