    We can’t use the above queries b’coz it might be the case that due to the update the condition of max age becomes false(update anomaly).Use the following query it works correctly.
    update table_name set name=’HIGH’ where age in ( select max(age) from (select * from table_name) as t group by type) and type in( ( select type from (select * from table_name) as s group by type having age=max(age)));

    UPDATE tab SET name=’HIGH’ WHERE name IN (SELECT name FROM tab WHERE age IN (SELECT max(age) FROM tab GROUP BY type))
