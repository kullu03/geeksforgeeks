    Given N X N matrix, row and column wise sorted.

    Median element

    ((n*n)+1)/2 if n is odd

    ciel(((n*n)+1)/2) or floor( ((n*n)+1)/2) if n is even (users choice) or it can be avg of ciel and floor

    Median is 13th element

    let i=0,j=0 // i is my column index and j is my row index

    loop{

    consider A[n-1,i] (Bottom Left) and A[j,n-1] (Top Right)

    if(A[n-1,i] > A[j,n-1]) then Mark all the elements A[j,i] to A[j,n-1], count them and j=j+1

    else i=i+1 and Mark all the elements from A[j,i] to A[n-1,i], count them and i=i+1

    Output the number when count == median.

    }

    On this example

    step1: i=0,j=0

    step2: compare (50,54) as 50 is smaller mark 10,20,30,40,50 (count =5) i=i+1

    step3: compare (54,91) as 54 is smaller mark 11,52,53,54 (count =9) j=j+1

    step4: compare (91,114) as 91 is smaller mark 21,31,41,91 (count =13) i=i+1

    as median is at count =13 therefore median = 91 (count=13)

    If N is odd, the median should be at M[N/2][ N/2].

    If N is even, the median should be (M[(N-1)/2][(N-1)/2] + M[(N-1)/2][(N-1)/2+1] + M[(N-1)/2+1][(N-1)/2] + M[(N-1)/2+1][(N-1)/2+1])/4.
