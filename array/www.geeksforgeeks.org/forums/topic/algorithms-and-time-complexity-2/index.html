    If you have an array of n elements, how would you devise an O(n lg n) time algorithm for outputting a list of those elements that appear more than once in the original array.

    2) Linearly scan the sorted array to print duplicate elements.

    Use an hashset to keep the elements.


    
ArrayList<Integer> list = new ArrayList<Integer>();
    
HashSet set = new HashSet<Integer>();
    
for(Integer i : list){
    
if(set.contains(i)){
    
System.out.println(i);
    
}else{
    
set.put(i);
    
}
    
}

    this approach takes O(n) in time, but uses O(n) space.
