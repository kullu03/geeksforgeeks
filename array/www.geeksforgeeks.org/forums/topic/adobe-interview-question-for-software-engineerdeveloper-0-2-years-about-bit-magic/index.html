    @Sandeep, I reopen the topic. The above posts are related questions, but not exact.

    It is good question. A stream can be stored on disk (serialized), and it may be required to test divisibility of a large number stored on the disk as binary stream by another number.

    The solution proposed by @sp is efficient one, but some correction required.

    Pseudocode


    
Input: Binary stream stored in a string "stream" and a number K (divisor).
    

    
weight = 1;
    
rem = stream[0] - '0'; // Assuming LSB in first position
    
index = 2;
    

    
do
    

    
int bit = stream[index] - '0';
    
weight <<= 1;
    
rem = ((bit == 1) * weight + rem) % K;
    

    
if(rem == 0)
    
// print stream received till now
    
index++
    
while(index < stream.length)

    Corner cases are not verified.
