    Maybe we could use this recurrence formula?

    C(0) = 1
    C(1) = 1
    C(n+1) = \sum\limits_{i=0}^n C(i)C(n-i)

    The idea is that you create chords between the points in such a way that you split the problem into two. For 2n points, the first splitting creates two problems each with 2(n-1) and zero points respectively, second splitting creates 2(n-2) and two points, third makes 2(n-3) and four points and so on.

    Select any one of the 2n points. No counting has taken place till now.

    How many ways can you draw one chord from that point which will divide the rest of the points into 2 even parts ?
    n

    Now we are done with that point and the point it joined and have 2 subproblems of similar nature.

    The solution is recursive and looks like below – each line is one subproblem – there are n lines:

    F(2n) =
    F(2n-2) * F(0) +
    F(2n-4) * F(2) +
    F(2n-6) * F(4) +
    ….
    F(0) * F(2n-2)

    From simple drawings you can see that F(0) = 1 ,
    F(2) = 1 (verify!),
    F(4) = 2 (verify!!),
    F(6) = 5 (verify!!!)

    I think answer is 2^(n-1). We can prove this by induction.

    i) Base case is true trivially(For n = 1, only one possible way)

    ii) Assume true for n points.

    iii) Now add two more points to the circle(2n+2 overall). Pick any point on the circle. In order to avoid intersection with other chords, this point should only be joined to the point immediately to the right of it or to the left of it. If it is joined to the point immediately to the right of it, there are now 2n points remaining and we already know that there are 2^(n-1) possible for this scenario. Similarly if it is jointed to the immediately to the left of it, there are again 2n points, for which 2^(n-1) possibilities are there. Therefore total no of ways = 2^(n-1) + 2^(n-1) = 2^((n+1)-1).
