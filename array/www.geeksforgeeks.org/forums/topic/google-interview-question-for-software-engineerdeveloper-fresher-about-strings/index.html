    Given an input string S write a function which returns true if it satisfies S = nT. Basically you have to find if a given string can be represented from a substring by iterating it “n” times. n >= 2

    An example would suffice –

    Function should return true if

    1) S = “abab”

    2) S = “abcdabcd”

    3) S = “abcabcabc”

    4) S = “zzxzzxzzx”

    Function should return false if

    1) S = “abac”

    2) S = “abcdabbd”

    3) S = “abcabcefg”

    4) S = “zzxzzyzzx”

    It would be easy to understand if you can give an algo instead of saying use kmp or suffix tree or…

    I came up with O(n*n) solution. Wondered how to do in O(n)

    This method can be used to check if a given string has a pattern. It is O(N) as it checks lengthOfString/2 possible patterns and quit from the loop as soon as 1st not matched pattern is found. Plus, it skips all checks when lengthOfString is odd and pattern number is even.


    
bool HasPattern(char *str, int& numPattern, char** substr)
    
{
    
bool hasPattern = false;
    
int len = strlen(str);
    
numPattern = 0;
    
int div = 2;
    
int k = len/div;
    
bool oddLen = (len%2 > 0 ? true : false);
    
char* buf1 = new char[k+1];
    
char* buf2 = new char[k+1];
    
*substr = NULL;
    

    
while(k > 1)
    
{
    
// To check if length of the string is odd. In this case we should skip all
    
// even values of div. It will reduce the number of lookups in 2 times
    
if(oddLen && (div%2 == 0))
    
{
    
div++;
    
k = len/div;
    
continue;
    
}
    

    
int l = 1;
    
// to copy 1st pattern into buf1
    
strncpy(buf1, str, k);
    
buf1[k] = '';
    

    
while(l < div)
    
{
    
// to copy every next (l's) pattern of length k into buf2
    
// if buf1 != buf2 -> don't match, break, and go to the next pattern
    
strncpy(buf2, str + k*l, k);
    
buf2[k] = '';
    

    
if(strncmp(buf1, buf2, k) != 0)
    
break;
    
l++;
    
}
    

    
if(l == div)
    
{
    
cout<<"The pattern is found."<<endl;
    
cout<<"Matched substring: "<<buf1<<endl;
    
cout<<"Pattern count: "<<div<<endl;
    
hasPattern = true;
    
numPattern = div;
    
*substr = new char[k+1];
    
strncpy(*substr, str, k);
    
(*substr)[k] = '';
    
break;
    
}
    
div++;
    
k = len/div;
    
}
    

    
delete[] buf1;
    
delete[] buf2;
    
return hasPattern;
    
}
    

    
int _tmain(int argc, _TCHAR* argv[])
    
{
    
char* str = "abcabcabc";
    
int numPattern;
    
char* substr = NULL;
    

    
bool match = HasPattern(str, numPattern, &substr);
    

    
str = "zxvzxvzxvzxvzxvzxv";
    
substr = NULL;
    
match = HasPattern(str, numPattern, &substr);
    

    
str = "zxvzxvzxvzxvzxvzx";
    
substr = NULL;
    
match = HasPattern(str, numPattern, &substr);
    

    
getchar();
    
return 0;
    
}

i++;

while(i<target.length())
    
{
    

    
if(target.charAt(i) == window.charAt(count))
    
{

{
    
count = 0;

{

}
    
else
    
{

}

      Your approach is not O(n), consider the worst-case input:
      aaaaaaaaabaaaaaaaaab

      Your approach will try with a window with ‘a’ until it fails when reading ‘b’, then it will try with ‘aa’ and so on. For this particular instance, it will pick n/2 different windows and make n/2..1 comparisons until it detects the non-matching character. Hence, it’s O(n^2) for worst-case inputs.

    I wouldn’t like having this question in an interview, because I think this is a kind of “Uh huh! This is the trick!” question.

    Let’s take “zzxzzxzzx” as an example. If the string can be written as nS, where n > 1, then its suffix is (n-1) * S. 

     ZZXZZXZZX 
     ZZXZZX

    Now, we need to add the S to the end somehow. We don’t know the actual length, so we add the whole original string instead, i.e. we take the original string twice.

    ZZXZZXZZXZZXZZXZZX

    Now, to get |S|, simply search for the position of the original string starting at the second character (string search is a common function in all languages). If it returns |original string|, it means it found the concatenated string and we simply return False. Otherwise, |S| != |original string|, therefore original string = n*S and we return True.

    This can be done in O(n) times. You would need to construct Border array for it complexity of which is O(n).
    Border of a string is any substring which both proper prefix and also suffix. Border array at index i would contain length of longest border for string ending at index i. KMP algorithm creates this for pattern to be matched. Here we would construct it for the given string itself. Pseudo code to create Border array B[] for sting S of length n:

    
B[0] = 0;
lastBorder = 0;
for (int i = 0; i < n; i++) 
{
  //Try to extend from i-1th largest border
  if(S[lastBorder + 1] == S[i]) {
      lastBorder ++;
      B[i] = lastBorder;
  } else {
  
      // As largest border at i-1 could not be extended
      // we will try to extend second largest, if not that
      // then third largest and so on.
      while (lastBorder >0 && B[lastBorder+1] == S[i]) {
          lastBorder = B[lastBorder];
      }
      if(S[lastBorder + 1] == S[i]) {
         lastBorder ++;
         B[i] = lastBorder;
      } else {
         // last border is zero
         B[i] = 0;
      }
  }
}

    Note: the above code can be optimized in code space but left for ease of understanding.

    Once it is created just check whether difference d = n – B[n-1] (last index) is divisor of ‘n’. If it is then return true else false.

    /*
first, we get the next array of pattern string.
second, check whether "len % (len-next[len])" equals zero. if so, return "len / (len-next[len])" > 1, otherwise false.
*/
void GetNext(const char *pattern, int next[])
{
    int i = 0;
    next[i] = -1;
    int j = next[i];
    while(*(pattern + i))
    {
        if(j == -1 || pattern[i] == pattern[j])
        {
            i++;
            j++;
            next[i] = j;
        }
        else
            j = next[j];
    }
}
bool isRepeats(const char *s, int next[])
{
    GetNext(s, next);
    int len = strlen(s);
    if(len %(len - next[len]) == 0)
        return len / (len - next[len]) > 1;
    return false;
}
