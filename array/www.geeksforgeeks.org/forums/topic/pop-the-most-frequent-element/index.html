stack1.push(item);
    
if (map.contains(item)) {

map.put(item, count+1);

map.put(item, 1);
    
}
    
}

    Maybe we can use just a hashtable along with a pointer/value of the most frequent element.

    
typedef struct elem_info {
  size_t count;
  elem_t prev_freq;
};

typedef struct freq_stack {
map<elem_t, elem_info> map;
elem_t most_freq;
}

    Every time we add an element we should update most_freq and prev_freq references appropriately. If it’s the same frequency always update most_freq to the latest added element. Then, when you remove the most_freq element we should fallback to the prev_freq element. Hope this makes sense, and works… should be all O(1).
