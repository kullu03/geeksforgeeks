    You have mobile phone. In a mobile phone, you are required to find a suitable data structure to store the names and search and retrieve them very efficiently. We can assume that there is very limited memory as it is an embedded device. Also we need to

    consider, if you key in say ‘A’ and if you have names Abhishek, Abhilash, Ashok all those should be retrieved and as you key in more characters, you must narrow down to the particular name you are searching for.

    What will be a suitable data structure for this.. I had started off with hash table, then went on to explain how binary search tree could be used and to using trees itself, and returning basically a subtree as and when more keys are typed.
