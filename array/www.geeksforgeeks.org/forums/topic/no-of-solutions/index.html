    A general solution for x1+x2+x3+x4+x5=n would be like this :

    To divide a no into 5 parts, the possible divisions are 2+3 or 3+2,

    now lets say, n=19 (the concept will be better explained with 19 rather than 20)

    19=9+10 (always divide as close to n/2 as possible)

    now this division can be of 9 types ie. 1+18, 2+17,…. 9+10.. thus equal to lowest number of the two nos.

    Now, recursively divide (9 into 2 parts * 10 into 3 parts) and again (9 into 3parts * 10 into 2 parts). Add the numbers. Summation of all these sets will give the result (ie 1+18, 2+17….).

    Similarly, for 3 part division divide like 2+1 and 1+2


    
int solution(int no, int div){
    
if(div==1)
    
return 1;                            //If one division is to be made, then only one possible solution
    
if(div==2)
    
return (floor(no/2));   //If 2 divisions are to be made, then lowest of the two is the solution ie 9 for 19
    
sum=0;
    
no1=floor(no/2);
    
no2=no-no1;
    
for(i=1;i<=no1;i++)        //Doing the calculations as mentioned above
    
sum+=(solution(i,(div/2))*solution(no-i,((div/2)+1)))+(solution(no-i,(div/2))*solution(i,((div/2)+1)));
    
return sum;
    
}
