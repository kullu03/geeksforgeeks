    answer is 9

    for visualisation draw a venn diagram

    english = 46 spanish =46 french = 58[these are not only values , these are people who may or may not speak other alnguages]

    english&spanish= 16

    spanish&french =24

    english&french=26

    Speakingallfour = 7

    now calculate only values for two languages[people who only speak two languages], subtract all four from these values

    Onlyenglish&spanish = 16-7 = 9

    Onlyspanish&french = 24-7=17

    Onlyenglish&french = 26-7=19

    now calculate only values for 1 language subtract only2 and all four

    onlyenglish = 46 -9-19-7=11

    onlyfrench = 15

    onlyenglish=13

    now add these only values[both single and double and all four and subtract from 100]

    answer is 9
