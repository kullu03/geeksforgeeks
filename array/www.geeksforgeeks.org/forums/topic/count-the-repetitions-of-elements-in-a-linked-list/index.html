    I have a linked list that contains an infinite number of integers linked together. I have to write a function that prints each integer and its count. For e.g, I have a list like

    [H 10 20 200] -> [A 200 20 30] -> [B 30 10 33] -> [C 33 100 13] -> [D 13 10 45] -> [E 45 10 1000] -> [F 1000 10 NULL]

    where
    [A 10 20 100] indicates A is the name for reference, 10 is the address where a is stored, 20 is the value, and 200 is the pointer to next node.

    I have to print the respective elements of the list and their counts:-
    Integer Count
    20 2
    10 4
    100 1

    What would be the best way to implement this?

    My Algo as I have thought is :-

    1) create a 2D array where we will be saving the unique elements and the respective counts.
    2) traverse the array for one element, if the element exists in the array for the first time, then keep on incrementing the value of count for that element in the second column of the array, second time we encounter the element, we should not process the element.
    3) keep on repeating the process for all the elements of the list and print it later.

    Problem with array is that it is finite and we have infinite number of elements and thus array becomes unsuitable.
    But I am thinking there should be some better feasible alternative, this one is more like baby coding and would be having more time complexity. Please suggest..
    Thanks in advance.
