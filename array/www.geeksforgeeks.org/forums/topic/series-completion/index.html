    Hello Saipreethi,

    My answer came out to be 4.

    Here’s how…

    Take any number bigger than given in this series, and subtract each digit given here from that number.

    The only missing link that we find gives the answer as 4.

    Example:

    bigger number = 9,

    7 = 9-4

    6 = 9-3

    8 = 9-1

    5 = 9-4

    3 = 9-6

    7 = 9-2

    Here, all digits (from 1 to 6 are subtracted from 9, except 5), but “9-5 = 4″ is not covered.

    Hence, my answer came out to be 4.

    I found it true for any number greater than 8 ( biggest number here).

    HTH.
