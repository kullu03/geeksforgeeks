    My Answer – [d]

    What is RAM?
    If you have more RAM computer can run more programs at same time.
    Types of Storage Devices:
    1. Permanent Memory [Hard disk/USB]: Holding data when electricity not there.
    2. Volatile Memory [RAM]: looses data when power not attached.
    Processor can only Run the program If it is on the RAM.
    At BOOT Time
    Processor runs those Instructions which are not Present on RAM(BIOS). It takes out OS(BIOS) from hard Disk and puts on RAM. OS is Booted.
    Running the Programs
    Bring the Program in RAM and run it.
    Paging
    the process which does this movement of task from Hard drive to RAM & back is called Pager.

    What is Virtual Memory?
    Problem:
    Our System has – 1MB of RAM
    1 Program needs – 10MB of space
    Solution:
    Virtual Memory.
    We can use HARD DISK for program. This memory (Hard Disk) is called Virtual since its not RAM, thought memory is present its not fake.
    But the problem with disk is – its really-really Slow.
