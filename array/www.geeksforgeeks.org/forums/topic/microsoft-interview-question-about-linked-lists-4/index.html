    Write a function that would: return the 5th element from the end in a singly linked list of integers, in one pass, and then provide a set of test cases against that function

  • April 5, 2010 at 1:53 AM #91125

    Anonymous

    0. A function that takes pointer to head and and an integer n, where n is the required position from last (in your case n = 5)

    1. Use two pointers, ‘current; and ‘ntolast’. Initially both point to head.

    2. Move ‘current’ to position n-th position from start by iterating n times. ‘ntolast’ should still point to ‘head’.

    3. ‘current’ and ‘ntolast’ are now n-1 elements apart.

    4. Start another loop, move ‘ntolast’ and ‘current’ by one position in each iteration till ‘current’ reaches end.

    5. Return ‘ntolast’ as answer once loop described in (4) exits.

    Certainly the above points will provide us the 5th to last element in one traversal…

    Here are some trivial test cases for the same :

    1. If the list is empty, it should give error.

    2. If the list has less than 5 elements, it should give an error.

    3. If the list has only 5 elements, it should provide us with the first element as the required answer.

    4. The last 1 is the normal case with more than 5 elements, where the correct element is returned accordingly.

    I can’t figure out any other test case for this problem. If anyone can think of another test case, then do let me know….
