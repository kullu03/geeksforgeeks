    Here is a python code. Raktim’s approach will work too.


    
f=open('inputfile.txt','r')
    
o=open('outputfile.txt','w')
    
f.seek(0,2) #Similar to the C fseek() function call, seeking to 0 byte from the end of the file
    
while f.tell()!=0:
    
f.seek(-1,1) #seek to one byte backwards from current position
    
o.write(f.read(1))
    
f.seek(-1,1)
    
f,close()
    
o.close()

    C++ Approach.

    Read file(input) in chunks.

    1. Read first chunk
    2. Reverse it.
    3. Push this in another file (output).
    4. Open output file in append mode and set put pointer to beginning of file seekp(ios::beg).
    5. First chunk will become last chunk and so on.
