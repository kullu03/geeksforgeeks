    1. Create DLL from the values of the array.

    2. Hash all the values with (value , DLL->nodeAddress);

    3. Start from beginning of the array and find all maximum span from the hashmap. if you find the value delete it from the DLL.

    ex: in above array first value is 1, you will look for 2(not found stop) and 0(not found stop) so current max span is [1], delete 1 from DLL.

    next element is 6, you will look for (7 found delete from DLL), 8(not found stop) and 5(found delete from DLL), 4(found delete from DLL), 3(not found stop). so current max span is [4-7] delete 6 from DLL. resultant DLL

    10->9

    we will follow the same process untill size of DLL >= current max span else return current max span

    The problem is available at leetcode.
    There exists a simple solution: (though the union-find set is a choice, but the solution posted here is a solution using hash.)
    The code is :
    int longestConsecutive(vector &num) {
    const int n = num.size();
    if(n < 2) return n;
    map lbound,rbound;
    int ret=0;
    for(auto x:num)
    {
    if(lbound.find(x) != lbound.end()) continue;
    lbound[x]=rbound[x]=x;
    //modify lbound
    if(lbound.find(x-1) != lbound.end())
    {
    lbound[x] = lbound[x-1];
    rbound[lbound[x]] = x;
    }
    //modify rbound
    if(rbound.find(x+1) != rbound.end())
    {
    rbound[x] = rbound[x+1];
    lbound[rbound[x+1]] = lbound[x];
    rbound[lbound[x]] = rbound[x+1];
    }

    ret = max(ret,rbound[x]-lbound[x]+1);
    }
    return ret;
    }
