    [n/2^(h+1)] ceiling function

    nodes of height h in any n-element heap

    Number of nodes Nh at any height h can be represented as (2^(H-h)) where H is height of the tree.
    Maximum value of total number of nodes Nt in a heap of height = 2^0+2^1+…+2^(H-h)+–+2^H [h=H to 0]
    Nt= 2^(H+1) – 1
    Nh=2^(H-h), Nt=2^(H+1)-1
    Solving these two equations gives Nh=(Nt+1)/(2^(H+1))
    => Nh=Ceiling(Nt/(2^(h+1)))
