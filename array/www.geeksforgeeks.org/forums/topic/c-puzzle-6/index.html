    What happen in the following two codes?

    Code 1:
    void fun(int *b)
    {
    int c=5;
    b=&c;
    (*b)++;
    }
    int main()
    {
    int a=4;
    int *p=&a;
    cout<<”previous value of a is “<<*p;

    fun(p);
    cout<<”after calling fun ,new value of a is “<<*p;
    return 0;}
    Code 2:

    void fun(int &b)
    {
    int c=5;

    b++;
    }
    int main()
    {
    int a=4;
    int &p=a;

    Code 1:

    previous value of a is 4
    after calling fun ,new value of a is 4

    Reason: p is a pointer and the address of a is assigned to it. but, in the function, the value of p is passed into another pointer b ( so, b is also pointing to a), but, locally, b is assigned with address of c.( Therefore, values of b and p is different, now.). So, even after modifying the value pointed by b, p and the value pointed by it will remain unchanged.

    Code 2:

    previous value of a is 4
    after calling fun ,new value of a is 5

    Reason: p is a reference variable, so, in symbol table, p and a will have same address. So, when p is passed in function, local variable b will be created with the same address assigned to it. So, any change in b will be reflected not only in b but also in p and a.
