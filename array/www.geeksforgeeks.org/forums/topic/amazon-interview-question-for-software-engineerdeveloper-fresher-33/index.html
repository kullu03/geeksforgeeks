    I think the following algo should also work ..
    It is easier to implement also

    i) Divide the given nos into three parts HEAD MID TAIL
    ii) Add 1 to nos HEAD MID
    (in case of carry, follow basic addition rules)
    iii) reverse the new HEAD(store it in HEAD_REV)
    iv) required ans is:- ‘new HEAD’ MID HEAD_REV

    Hoping that following example will help in better understanding of the algo

    let nos be:- 23469 9 12367
    So HEAD -> 23469 MID -> 9 TAIL –> 12367
    step 2:- 23469 9 +1 = 23470 0 (now HEAD -> 23470 MID -> 0 HEAD_REV -> 07432 )
    Ans:-
    23470 0 07432

    This algo can be extended to odd-length number

    IF any one finds any flaw in this procedure to inform me

package fifteendigitnextpalindrome;

import java.util.Scanner;
//works only for odd digit number
public class NextPalindrome {

	//method for reversing the head
	public static  int reverseInteger(int head) {
		int last_digit=0;
		int reverseHead=0;
		while (head != 0)
		{    
			last_digit = head % 10;   
			reverseHead = reverseHead * 10 + last_digit;
			head = head / 10; 
		}
		return reverseHead;
	}

	public static void main(String[] args) {
		Scanner scanner = new Scanner(System.in);
		System.out.println("Enter the number");
		String number = scanner.next();	
		int count=0;
		int h=0, m=0, t=0;
		String head = null, tail=null,mid=null;
		int num = Integer.parseInt(number);
		//calculating length of string
		while(num>0) {
			count++;
			num = num/10;	
		}

		head=number.substring(0, count/2);
		mid=number.substring((count/2), (count/2)+1);
		tail=number.substring((count/2)+1);
		//converting head, mid and tail from String to Integer
		h= Integer.parseInt(head);
		m= Integer.parseInt(mid);
		t = Integer.parseInt(tail);
		int reverseHead = reverseInteger(h);
		//Implementing the Algorithm for finding next Palindrome.
		if(reverseHead<=t) {
			if(m<=9) {
				m++;
			} else{
				m=0;
				h=h+1;
			}
		} 
		//Printing the next Palindrome
		StringBuilder nextPalindrome = new StringBuilder();
		nextPalindrome.append(h);
		nextPalindrome.append(m);
		nextPalindrome.append(reverseHead);
		String nextPalindromeString = nextPalindrome.toString();
		System.out.println("The next Palindrome ="+nextPalindromeString);
	}
}
