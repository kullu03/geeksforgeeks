    devide the given array ‘a’ into two array ‘b’ and ‘c’
    =>’b’ for all positive numbers of array ‘a’
    =>’c’ for all negative numbers of array ‘c’

    now sort both the arrays ‘b’ and ‘c’
    suppose size of array ‘b’=l and size of array ‘c’=m
    for maximum product
    max(b[l-1]*b[l-2]*b[l-3] , c[0]*c[1]*b[l-1])

    For the array { 9, 8, 5, -7, -6, -5, -4, -3, -2, -1 }
    array b={9,8,5}
    after sorting b={5,8,9}
    so l=3; where l>=3

    array c={-7,-6,-5,-4,-3,-2,-1}
    after sorting c={-7,-6,-5,-4,-3,-2,-1}
    so m=7;

    now maximum product
    max(b[l-1]*b[l-2]*b[l-3] , c[0]*c[1]*b[l-1])
    max(9*8*5, (-7)*(-6)*9)
    ans=378;
