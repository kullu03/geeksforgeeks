    Anonymous

    There is a similar problem with solution as follows: (source: http://placementsindia.blogspot.com/2007/12/solutions-to-few-google-top-interview.html)

    8.Given a Data Structure having first n integers and next n chars. A = i1 i2 i3 … iN c1 c2 c3 … cN.Write an in-place algorithm to rearrange the elements of the array ass A = i1 c1 i2 c2 … in cn

    Solution:we divide the array in four sections:[X,Y|A,B]

    It is easy to see that with swaps we can modify it to the form [X,A|Y,B].

    Now do recursion to solve [X|A] and [Y|B] separately,essentially using divide and conquer.[as given in comments section]

    For example:

    { 1,2,3,4,5,6,7,8,a,b,c,d,e,f,g,h }

    X = { 1,2,3,4 }, Y = { 5,6,7,8 }, A = { a,b,c,d }, B = {e,f,g,h}

    { 1,2,3,4,a,b,c,d,5,6,7,8,e,f,g,h }

    [X|A] = {1,2,3,4,a,b,c,d }

    U = { 1,2 }, V = {3,4}, W = {a,b}, Z = {c,d}

    swap V and W:

    [U,W|V,Z] = { 1,2,a,b,3,4,c,d }

    [U|W] = { 1,2,a,b }

    swap 2 and a:

    { 1,a,2,b }

    =======================

    Based on that solution, we can naively solve our problem as follows:

    First, convert it to a1, b1, a2, b2, …, an, bn, c1, c2, …, cn

    Next, convert the new array to the desired form a1, b1, c1, a2, b2, c2, …, an, bn, cn by considering (a1, b1) as A1, … (an, bn) as An.

    But I think there is a better way to do that.

    I came up with a formula to calculated destination index of an element on a particular index.

    Here is the pseudo code:
    1. start from index 1 (as index 0 element will be unchanged) and go till last but one element in array.
    2. Suppose an element is at index ind (lets say startIndex).
    3. calculate destination index of that element using following formula :
    destIndex= num * (ind%size) + ind/size
    where
    num is number of sublists (like in a1,a2,b1,b2 … 2 sublists)
    size is size of each sublist (2 in above example)
    ind is current index of the element.
    4. store arr[destIndex] into a temp variable and store a[ind] into a[destIndex]. So now a[ind] is at its correct place i.e. destInd.
    5. ind= destInd. (i.e. find out the correct place for value stored in temp.)
    6. Repeat step 3-5 until you reach to a point where destIndex= startInd. i.e. we reached from where we started in this iteration.
    7. go to step 1 (i.e.next iteration).
    8. If in any iteration, calculating desIndex from startIndex (only startIndex), gives you destIndex
