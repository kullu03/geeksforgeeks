    Let the original number be N.
    Get the position of the leftmost 1 bit in N(maxIndex).
    Create another integer M and set M(maxIndex – i) = N(i).

    N(i) = ith bit of N.
    M(maxIndex – i) = (maxIndex – i)th bit of M.

    After all bits are scanned.
    Take xor of M and N, if result is 0 then palindrome else not a palindrome.

    When an unsigned number is given a negative value a warning is issued by compiler and negative value is converted to equivalent unsigned number.

    In statement
    unsigned int y = -12;

    The value assigned to y = UINT_MAX + y(negative value) +1

    C/C++ automatically promotes a signed number to an unsigned number when a signed number is compared with unsigned number.

    In statement:
    if ( x > y)

    value of x is automatically promoted to unsigned value.
    Unsigned value of x = UINT_MAX + x (negative value) +1
