    A basic Idea:

    1) Find the occurrence of fist character of the word to be searched.

    2) Their are 3 cases, the character lies on the corners with 3 adjacent neighbors , on the edges with 5 adjacent neighbors and in interior with 8 adjacent.

    3) Based on the search information, we can construct a tree or TRIE structure of maximum size = strlen(word_to_be_searched) in which we can search .

    P. S. — Coding this idea may involve other complexities for sure.

    Finite Context Modelling Example.

    One probable answer:

    Entropy of character occurring with 1/2 probability = -lg(1/2) = 1

    Entropy of character occurring with 1/4 probability = -lg(1/4) = 2

    Similarly for other characters too.

    For a 256 character example

    the number of bits will be

    128 *1 + 64 *2 + 32 *3 + 16 *4 + 8*5+ 4 *6 + 2*7 + 1*8.
