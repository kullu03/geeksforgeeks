    You will need to parse the array 2 times. This fulfils the requirement that time complexity is O(N).

    In the first parse, count the number of negative integers, say

    X=number_of_negative_integers. Hence , the last index of a negative integer in the array will be X-1, and the starting index of positive integers will be X.

    After this, on the second parse of the aray, everytime you encounter a negative integer, increment the index and store. Every time you encounter a positive index, increment the index counter for positive numbers( whose initial value is set to x).

    Here is the program.

    int negative_counter=0;
    int positive_counter=X;(where x is the number of negative integers in the array)

    for (i=0; i {
    if (array[i]<0)//negative number
    {
    output_array[negative_counter]=array[i]);
    negative_counter++;
    }
    else if (array[i]>0)
    {
    output_array[positive_counter]=array[i]);
    positive_counter++;
    }
    }
