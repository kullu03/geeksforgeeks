    If we can destroy the list, then following is a very simple solution to detect loop.

    1) current = head
    
2) while (1)
    
{
    
if(current == NULL)
    
print "No Loop";  break;
    
next  = current->next;
    
if (next == head)
    
print "Loop Found"; break;
    
current->next = head;
    
current = next;
    
}

    Change next of every node to head. If we reach a node whose next is head, then the node is being visited again and there is a loop

  • February 17, 2012 at 7:28 AM in reply to: Given two strings print whether they are anagrams #111625

    geek4u
    Participant

    @Avinash: time complexity of your method is O(nlogn)

    You can also do it in O(n) time complexity.

    Let a and b be the two strings

    1) Compare sizes of both strings. If not same, then they are not anagrams.

    2) Let size be n.

    3) Create an array count[] of size 256 (Assuming the characters in both strings are stored using 8 bytes) and initialize all the values in count as 0.

    4) Run a loop for i = 0 to n-1

    ………a) count[a]++

    ………b) count[b]–

    5) If all entries in count[] are 0, then the strings are anagrams. Otherwise not.

    Time Complexiy: O(n)
