    Suppose, The points are P1,P2,P3….PN.

    1. Calculate the initial slope between P1 and P2,mark it as MaxSlope(MaxScope=Slope(P1,P2)) and Mark P1 as start point of maxslope(Max1P=P1 and Max2P=P2).

    2. For(int i=3;i<=N;i++)

    {

    if( slope(P[i-1],P) > MaxSlope )

    {

    if( Slope(MaxP,P) > slope(P[i-1],P) )

    {

    MaxSlope=Slope(MaxP,P);

    Max2P=P;

    }

    else

    {

    MaxSlope=Slope(P[i-1],P);

    Max1P=P[i-1];

    Max2P=P;

    }

    }

    }

    3. Max1P and Max2P are the two points referring to Maximum Slope.

    Time Complexity : O(N).
