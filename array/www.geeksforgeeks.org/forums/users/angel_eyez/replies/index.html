    A general solution for x1+x2+x3+x4+x5=n would be like this :

    To divide a no into 5 parts, the possible divisions are 2+3 or 3+2,

    now lets say, n=19 (the concept will be better explained with 19 rather than 20)

    19=9+10 (always divide as close to n/2 as possible)

    now this division can be of 9 types ie. 1+18, 2+17,…. 9+10.. thus equal to lowest number of the two nos.

    Now, recursively divide (9 into 2 parts * 10 into 3 parts) and again (9 into 3parts * 10 into 2 parts). Add the numbers. Summation of all these sets will give the result (ie 1+18, 2+17….).

    Similarly, for 3 part division divide like 2+1 and 1+2

    Use a circular queue for this problem say Q and two var pointing to start and end.

    1.take any arbitary point, considering that the point is anywhere in between the path.

    2.point start and end to the point. Initial petrol is the amount available in the pump selected.

    3.Now, if the truck can move with the available petrol to the next pump, point end to the next pump and refill and continue.

    4. If the available petrol isn’t sufficient, then iterate backward until the petrol becomes sufficient to move to next pump. Point start to the pump that fulfills the deficiency.

    5. continue to step 3 until start and end again points to the same pump.

    6. variable start thus stores the pump where the route should be started.

    For any consecutive addition , conditions possible are:

    1.n=(x-1)+( x)+( x+1)=3x, similarly 5x,7x,9x … ie ix i=3,5,7,…

    2.n=(x)+(x+1)=2x+1, similarly 4x+2,6x+3… ie ix+k i=2,4,6… and k=1,2,3…

    So, just maintain a loop and go on checking how many such possible sequence is possible

    Conditions:

    1. for odd cases , x should be greater than (i/2)

    2. for even cases, x should be greater than or equal to (i/2)

    3. x should be always integer not fraction.

    4. terminate the checking once case 1 or 2 fails

    e.g. n=9-> i = 2, x = 4 , ie 9=4+5

    i = 3, x = 3, ie 9=2+3+4

    i = 4, x = 7/4, so condition 3 fails
