    Since addition takes will place from last node, we cannot start from first node. To add carry towards beginning node follow these steps:

    1- Reverse first link list
    2- Reverse second link list.
    3- Now start from first node,write code for addition and take care of carry also.
    4- Go until either list ended and add carry (if it is) to other list node.
    5- Reverse the link list which represents final addition.

    1-first handle the first two node so that after swapping,you can excess element from head.
    2-Take three pointer, curr which holds current node, prev which hols previous of curr and nxt which hold next node of curr.
    3-link prev to nxt.
    4-curr to nxt->next
    5-nxt to curr.

    #### code ####

    struct node* Interchange(struct node *head)
    {
    struct node *curr;
    curr=head;
    struct node *nxt;
    struct node *prev;

    if(curr->next!=NULL)
    {
    head=curr->next;
    curr->next=head->next;
    head->next=curr;
    prev=curr;
    curr=curr->next;
    }
    while(curr!=NULL)
    {
    if(curr->next==NULL)
    return head;
    else
    {
    nxt=curr->next;
    prev->next=nxt;
    curr->next=nxt->next;
    nxt->next=curr;
    prev=curr;
    curr=curr->next;
    }
    }
    return head;

    }
