    In the previous sol, i had assumed that either one pair of consecutive Cs will be there or all the Cs will be sperated.

    below is the corrected solution 


    
int  count(int n, int bEncountered, int cConsecutive, int cWasLast, char arr[], int current)
    
{
    
int count1=0, count2=0, count3=0;
    
if(n<1)
    
{
    
arr[current]='';
    

    
printf("%sn", arr);
    

    
return 1;
    
}
    
else
    
{
    
arr[current]='a';
    
count1 = count(n-1, bEncountered, cConsecutive, 0, arr, current+1);
    

    
if( !bEncountered )
    
{
    
arr[current]='b';
    
count2 = count(n-1, 1, cConsecutive, 0, arr, current+1);
    
}
    

    
if( !cConsecutive )
    
{
    
arr[current]='c';
    

    
if( cWasLast )
    
{
    
count3 = count(n-1, bEncountered, 1, 1, arr, current+1 );
    
}
    
else
    
{
    
count3 = count(n-1, bEncountered, 0, 1, arr, current+1 );
    
}
    
}
    
else
    
{
    
if( !cWasLast )
    
{
    
arr[current]='c';
    
count3 = count(n-1, bEncountered, 1, 1, arr, current+1 );
    
}
    
}
    

    
return count1+count2+count3;
    
}
    
}

if(   firstNode->next != NULL  )//does second node exist?

    BFS requires a queue and to print in reverse we would need a stack.

    so, here using an array and two pointers TOP and FRONT. Elements will be always inserted to the index pointed by TOP.

    intially both TOP and FRONT would be pointing to index 0.

    1. insert the root node at TOP index and increment the TOP.

    2.get the FRONT node say it T (do not remove the element from the array) and increment FRONT.

    3.Inset children of T and increment Top .

    4.repeat step 2 and 3 untill FRONT<=TOP

    5.print all the elemnts from TOP to index 0.
