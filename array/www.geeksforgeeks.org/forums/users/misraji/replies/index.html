  • September 7, 2010 at 11:07 PM in reply to: Google Interview Question for Software Engineer/Developer about Algorithms, Arrays #98616

    Misraji
    Participant

    How about this:

    1. Perform a stable sort on the numbers. Thus by scanning this sorted array we

    will have the frequency of each number.

    2. Create a new array with each element being a pair (Number, frequency).

    The order should be the same as in the sorted array.

    3. Perform a stable sort on the array created in step 2, this time only on the frequency.

    We still maintain O(nlogn) time, and O(n) extra space.

    ~Ashish.
