    #algo:<adjacency list approach><naive algo>

    we will create node with

    struct node{

    char *s;

    struct node *right;

    struct node *down;

    }link;

    link node;

    1.initially pick the first string and make the node with char *s=string and right=down=null

    2.pick the next string and check with the first node.

    if these 2 are anagram then make newnode and link it to the right link of the first node

    else link it to the down link of the first node.

    thus we will loop on step 2 and either we will traverse right or traverse down and then will place it.

    thus we can cluster all the anagrams.

    next is very straighforward. first sort all those vertical nodes followed by sorting of horizontal nodes of each vertical nodes.
