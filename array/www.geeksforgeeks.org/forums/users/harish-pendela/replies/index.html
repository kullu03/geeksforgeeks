    1) Read one word at a time from dictionary and store it in a variable w
    2) Copy w to another variable c
    3) Now sort the characters in c
    4) Create a hash table with key as c.
    5) Now add w as value to the key c.
    6) If there is already a value to c append the new w to the existing value with a separator.
    7) Now we have a hash table where each key is the sorted form of a dictionary word and the value associated to it is a string of dictionary words that sorts to the same key.
    8) Repeat the process for every word in the dictionary.
    9) Now print values (which are anagrams) of each key as a group.
