  • Nitish kumar bharti

    Q. you have a billion numbers how do get kth top elements.

    Ans- Form a max heap tree from the given array and extract k elemnts from it.
    Overall complexity will be O(n)+O(klog n). which is O(n).

    • Tarzan

      Usage a min-heap is better as you will only need to compare with one element to know if the current element is in top-k elements

        • Tarzan

          Exactly, so you insert an element in the min-heap only if the current element is larger than the lowest element in the heap ( and if the heap-size is full then you call extract-min). That way you always maintain the l largest elements in the heap at any given time.

          Assuming you maintain a max-heap, you would then have to compare the current element with all the elements in the max-heap to decide if the current element is in top k largest elements
