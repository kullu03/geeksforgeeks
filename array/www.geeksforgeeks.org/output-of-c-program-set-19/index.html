Output of C Program | Set 19

Predict the outputs of following program.

Difficulty Level: Rookie

Question 1

#include <stdio.h>
int main()
{
  int a = 10, b = 20, c = 30;
  if (c > b > a)
  {
    printf("TRUE");
  }
  else
  {
    printf("FALSE");
  }
  getchar();
  return 0;
}

Output: FALSE
Let us consider the condition inside the if statement. Since there are two greater than (>) operators in expression “c > b > a”, associativity of > is considered. Associativity of > is left to right. So, expression c > b > a is evaluated as ( (c > b) > a ) which is false.

Hence the first time this statement is executed, the assignment operator sets the value of i to 0 and the post-incrementing operator increases it to 1. The next time it will be executed, it will assign the value of i as 1 first and then increase its value to 2 (again post-incrementing it). Hence, the loop will terminate when the value of i is > 6. This will result in printing the string value 5 times.
