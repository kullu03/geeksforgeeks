    • Abhimanyu

      Key thing here is the operator precedence.
      Operator precedence of *(indirection or pointer) is higher than that of +(arithmetic addition). So in expression
      *f1 = *f1 + (a +2.5)…*f1 will be evaluated 1st which is having value 2 and it will be added to 4.5(a+2.5) and result will be stored in &a (that is pointed by f1 and f2)
      In the second one result is as you expect it to be.

    • Nitish Kumar Maharana

      P1 :-> ans

      *f1 += ( a += 2.5 );
      => *f1 = *f1 + ( a = a + 2.5 )
      => *f1 = 2 + ( a = 2 + 2.5 ) ( here *f1 = 2, a = 2 given )
      => *f1 = 2 + 4.5
      => *f1 = 6.5

      P2 :-> ans

      a += ( a += 2.5 );

      => a = a + ( a = a + 2.5 )
      => a = a + ( a = 2 + 2.5 ) ( given a = 2 )
      => a = a + 4.5
      => a = 4.5 + 4.5 ( given a = 4.5 )
      => a = 8.5
